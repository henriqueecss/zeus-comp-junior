# Zeus — Sistema Interno Comp Júnior

## Meta do Produto
Fornecer à Comp Júnior um sistema web interno centralizado que permita 
gerenciar membros, squads, clientes e horas de forma eficiente, promovendo 
maior organização, transparência e controle sobre as operações internas 
da empresa.

## Critérios de Pronto Globais
Os critérios abaixo devem ser atendidos por qualquer issue antes de ser 
movida para a coluna "Concluído" no board.

### Desenvolvimento
- [ ] O código foi desenvolvido seguindo o stack definido (Node.js, React, MySQL)
- [ ] A funcionalidade foi implementada conforme a história de usuário
- [ ] Todos os critérios de aceite da issue foram atendidos
- [ ] Não há erros ou warnings no console

### Qualidade
- [ ] A funcionalidade foi testada manualmente pelo desenvolvedor
- [ ] A funcionalidade foi testada em dispositivo mobile
- [ ] Casos de erro foram tratados e exibem mensagens claras ao usuário

### Revisão
- [ ] O código foi revisado por outro membro da equipe
- [ ] A funcionalidade foi aprovada pelo Product Owner

### Entrega
- [ ] A funcionalidade está integrada com o restante do sistema
- [ ] O board no GitHub Projects foi atualizado para "Concluído"
